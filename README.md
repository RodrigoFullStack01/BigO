# Big O

Colecao de exercicios curtos em Python para praticar leitura de complexidade assintotica.
Cada arquivo resolve um problema pequeno e pode ser executado isoladamente.

## Arquivos

| Arquivo | Ideia principal | Ordem de crescimento |
| --- | --- | --- |
| `ex01.py` | acessar o primeiro elemento | `O(1)` |
| `ex02.py` | somar todos os itens da lista | `O(n)` |
| `ex03.py` | busca binaria em lista ordenada | `O(log n)` |
| `ex04.py` | encontrar pares que somam um alvo | `O(n^2)` |
| `ex05.py` | imprimir elementos e todas as combinacoes | `O(n^2)` |
| `ex06.py` | listar potencias de dois | `O(log n)` |
| `ex07.py` | fibonacci recursivo | `O(2^n)` |
| `ex08.py` | bubble sort | `O(n^2)` |
| `ex09.py` | multiplicacao de matrizes | `O(n^3)` |
| `ex10.py` | merge sort | `O(n log n)` |

## Como executar

```bash
python ex01.py
```

Troque o nome do arquivo para executar outro exercicio.

## Observacoes

- As respostas resumidas de complexidade estao em `ANALISE_BIG_O.md`.
- Os exemplos foram mantidos simples para destacar a logica e a ordem de crescimento.
